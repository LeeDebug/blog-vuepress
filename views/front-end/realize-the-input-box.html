    开发过程中遇到的新方案

    # textarea

    最常见的就是这种形式，不多赘述。代码如下

    但要展示带html元素的内容，可能就不是那么方便了

    <textarea
  id="sendMsgBox"
  class="sendMsgBox"
  ref="sendMsgBox"
  row="3"
  maxlength="1000"
  v-model="sendContent"
  :readonly="cantOperate"
  :placeholder="getPlaceholder"
  @blur="handleBlurFun"
  @focus="handleFocusFun"
  @keydown="handleKeyDown($event)"
  :style="{ 'height': textareHeight }"
></textarea>
